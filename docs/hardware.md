# Hardware

Initial target profiles:

- Raspberry Pi 5 with USB camera
- Raspberry Pi 4 with USB camera

Planned profile dimensions:

- board model
- camera type
- display or headless mode
- LED ring support
- cooling and throttling checks
- storage type and durability settings

## Recommended First Test Setup

- Raspberry Pi 5
- official power supply
- high-endurance microSD card or SSD
- USB camera
- wired network during early development
